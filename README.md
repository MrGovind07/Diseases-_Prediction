# 🩺 AI Disease Prediction System

This project is a Machine Learning-based web application that predicts whether a person is likely to have a disease based on health-related inputs such as age, BMI, blood pressure, glucose level, cholesterol, heart rate, smoking, alcohol consumption, physical activity, and family history.

The model is built using the **K-Nearest Neighbors (KNN)** algorithm and deployed with **Streamlit** to provide a simple and interactive user interface.

## ✨ Features
- Predicts disease using Machine Learning
- Interactive Streamlit web application
- Simple and user-friendly interface
- Real-time prediction based on user inputs

## 🛠️ Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 🚀 Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
